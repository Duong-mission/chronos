## Công nghệ sử dụng

Dự án được xây dựng dựa trên kiến trúc **MVVM** và **Clean Architecture** cơ bản.

*   **Ngôn ngữ:** 100% Dart.
*   **Framework:** Flutter (Cross-platform).
*   **State Management:** Provider (`ChangeNotifier`, `MultiProvider`).
*   **Cơ sở dữ liệu (Local):** **Isar Database** (NoSQL, siêu nhanh, hỗ trợ Full-text search).
*   **AI Integration:** Google Generative AI SDK (Gemini Flash Model).
*   **UI/UX & Graphics:** Custom RenderObjects (MindMap), fl_chart, Glassmorphism & Neon Theme.

## Điểm nhấn kỹ thuật

*   **Custom RenderObjects:** Tự xây dựng Widget `TreeBranchLayout` vẽ sơ đồ tư duy hiệu năng cao.
*   **Reactive Data Flow:** UI tự động cập nhật (Real-time) với `watchLazy()` của Isar.
*   **AI Streaming:** Phân rã công việc theo cấu trúc cây (Nested Tree) bằng Prompt Engineering chuyên biệt.

## Cài đặt và chạy thử

1.  **Clone repository:** `git clone https://github.com/your-username/chronos-flutter.git`
2.  **Cài đặt dependencies:** `flutter pub get`
3.  **Cấu hình API Key:** Cập nhật key trong `lib/core/constants/app_keys.dart`.
4.  **Chạy ứng dụng:** `flutter run`
